# ¿Qué es TRIAGE OS?

TRIAGE es un sistema operativo para orquestar agentes de IA especializados.

## Cómo funciona
- Tu tarea → TRIAGE clasifica → Elige qué agentes activar
- Code Agent (código + tests)
- QA Agent (bugs, seguridad)
- Research Agent (best practices)
- Risk Agent (impacto, rollback)
- Ejecutan herramientas reales → Validation Gate → Aprende

## Lo que lo hace especial
1. **Valida contra realidad** (no confía en predicciones)
2. **Aprende automáticamente** (patterns + blocklist)
3. **Bloquea patrones peligrosos**
4. **78% ahorro de tokens**
