## 🎯 Descripción General

**Sistema universal de 29+ Claude Skills** documentados profesionalmente, listos para instalar en tu entorno y usar inmediatamente. Cobertura total de productividad, contenido, diseño, desarrollo y automatización.

**¿Qué es un Skill?** Una metodología estructurada que activa un modo de trabajo específico en Claude para mejorar la calidad del resultado final.

## 🚀 Quick Start

### 1. Instalar Todos los Skills (2 minutos)

```bash
# Clonar este repositorio
git clone https://github.com/fsosnik/skills.git
cd skills

# Ejecutar instalador universal
bash install-all-skills.sh

# Verificar instalación
ls ~/.claude/skills/
```

### 2. Usar un Skill (Inmediato)

**En Claude.ai:**
```
"Activa Superpowers y construye una aplicación que..."
"Humaniza este texto..."
"Verifica este post antes de publicar"
```

**En Claude Code:**
```bash
# Los skills están disponibles automáticamente
```

**Desde tu propia máquina:**
```bash
# Los skills instalados en ~/.claude/skills/ se cargan automáticamente
```

### 3. Buscar Skills (Con Find-Skills)

```bash
npx skills find react  # Buscar skills relacionados con React
npx skills find testing  # Buscar skills de testing
```

## 🔄 Cómo Funcionan los Skills

### Paso 1: Activación
```
Usuario en Claude: "Activa [Skill] y..."
→ Claude carga el contenido del SKILL.md
→ Entra en modo especial de trabajo
```

### Paso 2: Ejecución
```
Claude sigue el proceso definido en el skill
→ Aplica la metodología estructurada
→ Mejora significativamente la calidad del resultado
```

### Paso 3: Entrega
```
Resultado final aplicando la metodología
→ Mejor que sin usar el skill
→ Reproducible y consistente
```
