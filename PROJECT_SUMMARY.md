# TRIAGE OS — Project Summary

## What We Built
Agentic operating system for Claude decision-making via:
- 4 parallel specialists (Code, QA, Research, Risk)
- Reality-driven validation (npm test, npm build, curl prod)
- Self-learning (weight updates, blocklist, pattern library)
- Token optimization (14,612 → 4,000 via Graphify)
- Production deployment (Fedora + PM2)
